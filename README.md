# SwpRequest

##### 导入：

```
手动导入：

SwpRequest 文件夹 导入 项目 中, 依赖  AFNetworking3.0 以上版本, Base64
#import "SwpRequestHeader.h"
```
---

```ruby

CocoaPods 导入:
使用  CocoaPods 会自动 pod AFNetworking
pod search SwpRequest

pod 'SwpRequest'

#import <SwpRequest/SwpRequestHeader.h>

```
